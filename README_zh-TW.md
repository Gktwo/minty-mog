# M
「原神」的新動態載入工具

## 搭建說明
1. 下載Visual Studio社群2022
2. 建置為 x64 版本。 不以調試為目標，也不以 x86 或 Win32 為目標。
3. 開啟解決方案的檔案總管時，您將發現一個名為「x64」的資料夾。 在那裡，您可以找到一個名為“Release”的資料夾，其中包含已建置的檔案。
